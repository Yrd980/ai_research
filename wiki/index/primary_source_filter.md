# Primary Source Filter

用于约束名词抽取与共现索引的来源质量。

## 目标

- 仅基于日报原文中的可追溯链接与条目文本提取名词。
- 降低转载、评论帖与噪声来源影响。

## 入选规则

1. 名词必须来自 `data/raw/wechat/*.md` 的正文条目。
2. 对应条目在原文中具备可追溯链接或上下文。
3. 名词应保持最小不可再分，避免将整句当实体。
4. 共现边仅来自同一条目内部，不跨条目拼接。

## 排除规则

- 不把主观判断词、情绪词、营销词写入 `terms.csv`。
- 不把日报转述语句直接复制成 wiki 断言文本。
- 不创建独立的 assertion 审核/晋升链路。
