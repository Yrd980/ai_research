# WeChat 原文入库目录

这个目录只放「原文层」数据，不做清洗、不做结构化改写。

## 命名规范

- 每日一文件：`YYYY-MM-DD.md`
- 示例：`2026-02-11.md`

## 入库原则

- 只保留原文与原始链接，避免二次加工污染。
- 明确输入渠道（手动粘贴/链接转录等）。
- 若当天文本未完整落盘，必须标记 `status: pending_sync`。

## 建议流程

1. 先把原文贴入当日文件。
2. 再更新 `ingest_manifest.csv` 的状态。
3. 最后只做 `data/processed/primitives.csv` 的原语抽取更新。
