## 一、世界模型主线（五层）

## 1) 技术层（模型能力怎么变）

### 发生了什么（事实）
- 图像：`Qwen-Image-2.0`、`Grok Imagine Image Pro`
- 语音：`MOSS-TTS` 家族
- 文本/推理：`LLaDA2.1`、`HY-1.8B-2Bit`、`Composer 1.5`
- 视频：`Sora Extensions`

### 为什么是结构变化（关联）
- 不再是单一模型参数竞赛，而是“多模态 + 可控编辑 + 长任务执行”并进。
- “生成”与“编辑”在同一模型中合流，意味着模型正从创作工具向生产工具演进。

### 处于哪一环（位置）
- 处在“能力商品化前夜”：能力已经足够强，下一步瓶颈转向基础设施与入口。

## 2) 协议层（模型如何接入世界）

### 发生了什么（事实）
- `MCP`（draw.io 官方服务）、`Gemini skills`、`ModelScope OpenAPI`
- `Responses API` 扩展长任务、容器、skills 接口
- `WebMCP` 在 Chrome 146 出现预览

### 为什么是结构变化（关联）
- 协议层的集中出现，说明行业正在把“调用模型”升级为“调用能力网络”。
- 协议不是附属功能，而是未来平台控制权的接口边界。

### 处于哪一环（位置）
- 处在“标准竞争初期”：多协议并存，生态绑定开始发生。

## 3) 入口层（用户和开发者从哪里进入AI）

### 发生了什么（事实）
- Chat 入口：ChatGPT 广告测试、Deep Research 升级
- IDE/开发入口：GPT-5.3-Codex 向 Cursor/VS Code/Copilot 扩展
- 桌面入口：`WorkBuddy`、`Claude Cowork (Windows)`

### 为什么是结构变化（关联）
- 入口形态从“单一聊天框”转向“聊天 + IDE + 桌面工作台”并行。
- 用户触点越接近工作流，平台越能沉淀行为数据与商业化能力。

### 处于哪一环（位置）
- 处在“入口重构阶段”：入口竞争已从产品体验升级为生态位争夺。

## 4) 资本层（资源如何重新配置）

### 发生了什么（事实）
- 融资：`Entire`（6000万美元种子）、`Runway`（3.15亿美元E轮）
- 并购：`Nebius` 收购 `Tavily`

### 为什么是结构变化（关联）
- 资本重点指向“可嵌入工作流的基础能力”（会话治理、检索验证、生成生产化）。
- 并购信号说明“关键能力模块”正在被云平台内化。

### 处于哪一环（位置）
- 处在“能力整合期”：基础模块由独立工具向平台能力迁移。

## 5) 组织层（公司如何重构协作）

### 发生了什么（事实）
- `Entire` 把 Agent 会话纳入 Git 工作流
- `Warp Oz` 强调可编排、可审计的 Agent 运行
- `CatPaw`、`Claude Code` 都在推进开发流程内嵌

### 为什么是结构变化（关联）
- 真正被重构的不是“写代码速度”，而是“组织如何与AI协作、追踪、治理”。
- 从“人使用工具”转向“人编排系统、系统协同完成任务”。

### 处于哪一环（位置）
- 处在“组织操作系统变更期”。

## 二、跨层因果链（只基于现有事实）

`多模态能力成熟`
→ `需要稳定接入与执行`
→ `协议层与API层加速建设`
→ `入口形态转向工作流场景`
→ `资本流向可整合模块`
→ `组织协作方式改变`

这条链条在三天样本中已经形成闭环证据（技术、协议、入口、资本、组织五层均有对应事件）。

---

## 三、格局中的“对抗轴”

- 轴1：**能力轴**（谁的模型更好） vs **系统轴**（谁的流程更可落地）
- 轴2：**开放生态**（OpenAPI/skills/MCP） vs **平台内化**（并购/闭环分发）
- 轴3：**免费增长逻辑**（Free/Go扩展） vs **商业化逻辑**（广告/配额/路由策略）

这三条对抗轴共同定义了当前 AI 时代变革中的主要张力。
